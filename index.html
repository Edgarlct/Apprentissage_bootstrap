Base

Container

La class container est prédéfini elle donne width prédéfini et s'adapte.
La class container peut prendre un breackpoint. La taille max d'applique donc seulement à partir de ce breakpoint.

class name Dimensions
None <576
sm ≥576px
md ≥768px
lg ≥992px
xl ≥1200px
xxl ≥1400px 
<div class="container-sm"><div>
<div class="container-md"><div>
<div class="container-lg"><div>
<div class="container-xl"><div>
<div class="container-xxl"><div>

Le container fluid a une width de 100% pour tout les breakpoints 

<div class="container-fluid"><div>

Marge

Le margin et padding ce règle aussi depuis des classes.
m pour le margin et p pour le paddin

class name Direction
t top
b bottom
l left
r right
x left and right
y top and bottom

Les valeur vont de 1 à 5 et auto 

<div class="mt-5"><div>

La taille des colonnes

Il est possible de dimentionner la taille des colonnes, si vous dimentionner la taille d'une seul les autres prendrons le reste de place disponible.

Colonne
Colonne taille définie
Colonne

<div class="container">
    <div class="row">
        <div class="col">
            Column
        </div>
        <div class="col-6">
            Colonne taille définie
        </div>
        <div class="col">
            Column
        </div>
    </div>
</div>

Il est possible de faire des colonnes avec une taille qui s'adapte au contenu 

<div class="col-{breakpoint}-auto">(md, ls, xl, xxl) Width varible <div>
