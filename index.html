Container

La class container est prédéfini elle donne width prédéfini et s'adapte.
La class container peut prendre un breackpoint. La taille max d'applique donc seulement à partir de ce breakpoint.
class name Dimensions
None <576
sm ≥576px
md ≥768px
lg ≥992px
xl ≥1200px
xxl ≥1400px 
<div class="container-sm"><div>
<div class="container-md"><div>
<div class="container-lg"><div>
<div class="container-xl"><div>
<div class="container-xxl"><div>
Le container fluid a une width de 100% pour tout les breakpoints 
<div class="container-fluid"><div>
